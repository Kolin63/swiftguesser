Rules

  • Guess the Taylor Swift Song as swiftly as possible!
  • In the game, press play to start the song (or ctrl+space)
  • Pause it as soon as you can figure out what song it is
  • Search for the song in the search box

Config

  • Swift Guesser has an extensive Config System
  • Customize your game by disabling albums, or adding more challenge
  • The Config you select will place you in different Leaderboard Rankings

Leaderboard

  • Swift Guesser features an Arcade-Style Leaderboard
  • Set your 3-character name to something that identifies you
  • Try to get into the top 10 for every song!
